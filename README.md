# HRM Experiments

A Python project for HRM experiments.

## Getting Started

Run the main script:
```bash
python main.py
```

## Project Structure

- `main.py` - Main entry point
- `.gitignore` - Git ignore rules (configured for Python and ML artifacts)

## Development

This project is set up for Python development with support for:
- Virtual environments
- PyTorch models (*.pt)
- Safetensors format
- Model checkpoints
